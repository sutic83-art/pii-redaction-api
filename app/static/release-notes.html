Current Limitations

Intentionally Not Included in V1

  • No database - file-based storage for simplicity and reliability
  • No user accounts - API key based authentication only
  • No real-time processing - batch processing for optimal accuracy
  • No custom model training - uses pre-trained Serbian language models
  • No document processing - text-only API (no PDF/Word/image support)
  • No streaming API - request/response pattern only
  • No multi-tenant UI - single deployment per customer
